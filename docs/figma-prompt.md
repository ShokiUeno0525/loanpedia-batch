# Figma デザイン生成プロンプト

## サービス概要

**サービス名**: Loanpedia（ローンペディア）

**コンセプト**: 青森県内の金融機関のローン情報を集約し、ユーザーが簡単に検索・比較できる無料の公共サービス

**ターゲットユーザー**: 青森県内でローンを探している一般消費者（住宅、マイカー、教育、リフォームローン等）

**提供価値**:
- 複数の金融機関サイトを巡回する手間を削減
- 条件で絞り込んで自分に適したローンを発見
- AI要約でわかりやすい情報提供
- 完全無料・ユーザー登録制

---

## デザインの方向性

**トーンアンドマナー**:
- 信頼感・安心感を与える落ち着いたデザイン
- 金融サービスらしい誠実さ
- 専門知識がなくても使いやすいシンプルなUI
- 地域密着型サービスとしての温かみ

**カラーパレット**:
- プライマリ: 青系（信頼感、誠実さ）
- セカンダリ: グレー系（落ち着き）
- アクセント: 温かみのあるオレンジ系（親しみやすさ）

**タイポグラフィ**:
- 視認性の高いゴシック系フォント
- 数字（金利、金額）は強調

## 画面一覧（全9画面 + 4コンポーネント）

### 1. S00-01: ランディングページ（初期表示画面）

**目的**: サービスの価値を伝え、ログイン・新規登録に誘導

**必須要素**:
- **ヒーローセクション**
  - キャッチコピー: 「青森県のローン情報を一箇所で」
  - サブコピー: 「4つの金融機関のローン商品を無料で検索・比較」
  - CTAボタン（大）: 「無料で始める」「ログイン」
- **価値提案セクション**
  - アイコン + 見出し + 説明文 × 4つ
    1. 時間の節約
    2. 検索・絞り込みの容易さ
    3. 最新情報の入手
    4. わかりやすいAI要約
- **対象金融機関セクション**
  - 4つの金融機関名とロゴ（プレースホルダー可）
- **使い方セクション（簡易）**
  - ステップ1: 無料登録
  - ステップ2: 条件で検索
  - ステップ3: 詳細確認
- **フッター**: 利用規約、プライバシーポリシーリンク

### 7. S03-01: ローン商品一覧画面（メイン画面）

**目的**: ローン商品の検索・絞り込み・一覧表示

**必須要素**:
- **検索・フィルタエリア（左サイドバーまたは上部）**

  - 金融機関フィルタ（チェックボックス × 4）
  - ローン種別フィルタ（住宅、車、教育等）
  - 金利範囲スライダー（最低〜最高）
  - 融資額範囲スライダー
  - 返済期間範囲スライダー
  - 「検索」ボタン、「クリア」ボタン
- **ソート機能**
  - ドロップダウン: 金利順、融資額順、更新日順
  - 昇順/降順トグル
- **商品一覧テーブル（またはカードリスト）**
  - 各行/カードの表示項目:
    - 金融機関名
    - 商品名（クリック可能）
    - 最低金利〜最高金利
    - 融資額範囲
    - 返済期間範囲
    - お気に入りボタン（ハートアイコン）
  - ページネーション（下部）
- **結果件数表示**: 「120件のローン商品」

## レスポンシブ対応

**ブレークポイント**:
- Desktop: 1200px以上
- Tablet: 768px〜1199px
- Mobile: 767px以下

**対応方針**:
- Desktop: 3カラムレイアウト可能
- Tablet: 2カラムまたは1カラム
- Mobile: 1カラム、ハンバーガーメニュー
