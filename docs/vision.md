# プロダクトビジョン

## 解決したい課題

青森県内の金融機関が提供するローン情報は、各金融機関のWebサイトに点在しており、ユーザーが複数の金融機関のローン商品を比較検討する際に以下の課題があります：

- **情報収集の手間**: 各金融機関のサイトを個別に訪問し、情報を探す必要がある
- **比較の困難さ**: 金利、融資額、返済期間などの条件を横並びで比較できない
- **情報の見落とし**: 全ての金融機関を網羅的に確認するのが難しく、最適な選択肢を見逃す可能性がある
- **理解の難しさ**: 金融専門用語が多く、各商品の特徴を理解しにくい

## ターゲットユーザー

**青森県内でローンを探している一般消費者**

- 住宅ローン、マイカーローン、教育ローン、リフォームローンなど、様々なローンニーズを持つ個人
- 金融知識のレベルは問わず、幅広い層を対象とする

## 提供価値

本システムは、以下の価値をユーザーに提供します：

### 1. 時間の節約

複数の金融機関サイトを巡回する手間を省き、一箇所で全ての情報を確認できます。

### 2. 検索・絞り込みの容易さ

金利、融資額、返済期間などの条件で絞り込み、自分に適したローンを見つけやすくします。

### 3. 最新情報の入手

月次で各金融機関の情報を自動収集・更新することで、常に最新の情報を提供します。

### 4. わかりやすい説明

AI技術を活用してローン情報を要約・構造化し、専門知識がなくても理解しやすい形で情報を提供します。

## 対象スコープ

### 対象地域

- **青森県内**

### 対象金融機関

| 金融機関名 | 機関コード |
|------------|------------|
| 青森みちのく銀行 | 0117 |
| 青い森信用金庫 | 1250 |
| 東奥信用金庫 | 1251 |
| 青森県信用組合 | 2260 |

### 対象ローン種別

- 住宅ローン
- マイカーローン
- 教育ローン
- リフォームローン
- その他各金融機関が提供する個人向けローン商品

### 提供機能

1. **ローン情報の集約表示**: 複数金融機関のローン商品を一覧で表示
2. **検索機能**: ローン種別、金融機関、条件などで絞り込み検索
3. **AI要約**: 各ローン商品の特徴をわかりやすく要約
4. **お気に入り機能**: 気になる商品を保存して後から確認

## 運営方針

本システムは**完全無料の公共サービス**として提供します。

- 広告表示なし
- 利用料金なし
- ユーザー登録必須（無料アカウント作成）
- 地域の金融情報へのアクセス向上を目的とした公益性の高いサービス

## システムの制約

### 情報の性質

- 本システムは各金融機関のWebサイトから公開情報を収集して提供します
- **申込機能は提供しません**：ユーザーは情報を確認した後、各金融機関に直接問い合わせる必要があります
- 情報の正確性は月次更新により維持されますが、最終的な判断は各金融機関への確認を推奨します

### 技術的制約

- データ更新頻度：月次バッチ処理
- 対象金融機関：現時点で4機関（将来的な拡張可能性あり）
